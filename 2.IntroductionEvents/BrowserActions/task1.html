Почему в коде ниже return false не работает? 


        <script>
            function handler() {
                alert( "..." );
                return false;
            }
        </script>
      
        <a href="https://w3.org" onclick="handler()">браузер откроет w3.org</a>

Браузер переходит по указанной ссылке, но нам этого не нужно.
Как поправить?

Ответ: потому что в приведенном примере обработчик событий делает вокруг функции функцию-обертку и не возвращает false. Нужно либо в обработчике событий возвращать функцию, тоесть написать onclick="return handler()", либо использовать preventDefault

Исправленный вариант

браузер откроет w3.org
