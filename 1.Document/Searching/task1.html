Searching: getElement*, querySelector*

1. Вот документ с таблицей и формой.
Как найти?…

  1. Таблицу с id="age-table".
  2. Все элементы label внутри этой таблицы (их три).
  3. Первый td в этой таблице (со словом «Age»).
  4. Форму form с именем name="search".
  5. Первый input в этой форме.
  6. Последний input в этой форме.

Ответ:

  1. document.getElementById('age-table')
  2. document.getElementById('age-table').getElementsByTagName('label')
  3. document.getElementById('age-table').querySelector('td')
  4. document.querySelector('form[name="search"]')
  5. document.querySelector('form[name="search"]').querySelector('input')
  6. let inputs = document.querySelector('form[name="search"]').querySelectorAll('input')
